# Jie Wu | AI Algorithms

Static personal website for GitHub Pages, focused on AI algorithms, machine learning systems, evaluation, retrieval, and research notes.

## Preview Locally

Open `index.html` directly in a browser, or run a small static server:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Publish to GitHub Pages

1. Create a GitHub repository named `<your-github-username>.github.io`.
2. Commit and push these files to the `main` branch.
3. In GitHub, open `Settings` -> `Pages`.
4. Use `Deploy from a branch`, then select `main` and `/root`.
5. Visit `https://<your-github-username>.github.io` after GitHub finishes deploying.

## Customize

- Replace `hello@example.com` in `index.html`.
- Replace project cards with your real AI/ML projects.
- Replace notes with real posts, papers, or technical writeups.
- Replace `assets/hero-ai.png` with your own AI research visual if desired.
